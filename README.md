## Installation
1. Clone repository
   ```bash
   git clone https://github.com/HaeengIn/school-pathfinder.git && cd school-pathfinder
   ```
2. Install networkx via pip
   ```bash
   pip install networkx
   ```
3. Run `main.py`
   ```bash
   python main.py
   ```
## Contribution
1. Clone repository
   ```bash
   git clone https://github.com/HaeengIn/school-pathfinder.git && cd school-pathfinder
   ```
2. Install required packages via uv
   ```bash
   uv sync --frozen
   ```
